Struct NonNativeUintVar

Source 
pub struct NonNativeUintVar<F: PrimeField>(pub Vec<LimbVar<F>>);
Expand description

NonNativeUintVar represents a non-native unsigned integer (BigUint) in the circuit. We apply xJsnark’s techniques for efficient operations on NonNativeUintVar. Note that NonNativeUintVar is different from arkworks’ NonNativeFieldVar in that the latter runs the expensive reduce (align + modulo in our terminology) after each arithmetic operation, while the former only reduces the integer when explicitly called.
